Limbaje:

  • HTML

    Este limbajul fundamental folosit pentru structurarea conținutului unei pagini web, definind elemente precum titluri, text, imagini și linkuri.

  • CSS

    Permite stilizarea elementelor HTML și controlează aspectul vizual al site-ului web, inclusiv culorile, fonturile, aspectul și efectele grafice.

  • JavaScript

    Un limbaj de programare care adaugă interactivitate paginilor web, permițând acțiuni dinamice și actualizări de conținut în timp real.

  • Node.js

    Un mediu de execuție care permite JavaScript să ruleze pe partea de server, permițând o dezvoltare backend rapidă, scalabilă și eficientă.

  • Express.js

    Un framework web ușor pentru Node.js care simplifică construirea API-urilor și a aplicațiilor server-side cu funcții puternice de rutare și middleware.

  • MongoDB

    O bază de date NoSQL care stochează date în documente flexibile, de tip JSON, facilitând gestionarea seturilor de date mari, nestructurate și în continuă evoluție.

  • PHP

    Un limbaj de programare pe partea de server utilizat pentru generarea de conținut dinamic și interacțiunea cu bazele de date în aplicațiile web.

  • MySql

    Un sistem de gestionare a bazelor de date relaționale utilizat pentru stocarea și organizarea eficientă a datelor unui site web.
